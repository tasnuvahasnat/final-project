Stuy Students against Yellow Minions (SSAYM), 2021

Game Development

How the Game Works

  • random: We used random to supply a random event to the player each turn.
  • matplotlib: matplotlib was used to create graphs of the status for various global variables like: wealth, morale, and troop size.
  • turtles: We used turtles to create an interactive map that changes based on the player's actions during the game.
  • lists and dictionaries: All the events and possible outcomes were stored in dictionaries. This allowed us to access it through random and program status changes as the player progressed. We used lists to monitor status changes and add them to the status graphs.
  • input: Our game implemented input for the player's name, empire, and each decision made.
